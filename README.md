# Project Title
- Dungeon Framework

## Description
- A dungeon explorer with randomly generated rooms from a premade pool. Solve puzzles and collect loot and sell it to move faster through the dungeon.

---

# How to Play
- Head over to "https://riftworks-studios.itch.io/dungeon-framework"

## Objective
- Wander the dungeon and discover new rooms.

## Basic Gameplay
- Enter new rooms through doorways
- Use levers/pressure plates to open doorways or unlock chest
- Collect loot and leave the dungeon to trade it in for coins
- Use coins to enhance your movement speed to traverse the dungeon in a speedier fashion

## Controls
- **W** / **Up Arrow** – Move up  
- **A** / **Left Arrow** – Move left  
- **S** / **Down Arrow** – Move down  
- **D** / **Right Arrow** – Move right  
- **F** – Activate (e.g., levers, pick up items)
- **Scroll** - Change selected item in the inventory
